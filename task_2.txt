# Git: Работа с репозиторием DockerGitCourse

## Добавление репозитория на GitHub
# Создаем репозиторий на GitHub с именем DockerGitCourse (пустой)

# Подключаем удаленный репозиторий
git remote add origin git@github.com:<USERNAME>/DockerGitCourse.git

# Переименовываем ветку в main
git branch -M main

# Отправляем локальные коммиты на GitHub
git push -u origin main

---

## Клонирование репозитория
# Удаляем локальный репозиторий (для проверки клонирования)
cd ..
rm -rf DockerGitCourse

# Клонируем репозиторий с GitHub
git clone git@github.com:<USERNAME>/DockerGitCourse.git

# Переходим в директорию проекта
cd DockerGitCourse

# Смотрим содержимое
ls -la

---

## Получение изменений с GitHub
# Перед началом работы забираем изменения
git pull --rebase

# Проверяем статус
git status
