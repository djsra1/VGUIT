Отчет по работе с Git – Задание 14: Открытые проекты

1. Клонирование репозитория:
git clone <URL_репозитория>

2. Создание новой ветки для изменений:
git checkout -b my-first-change

3. Просмотр состояния репозитория:
git status

4. Добавление изменений в индекс:
git add .

5. Фиксация изменений:
git commit -m "Первое изменение в ветке my-first-change"

6. Отправка ветки на GitHub:
git push origin my-first-change

7. Создание Pull Request:
- Перейти по ссылке, предложенной GitHub после git push
- Заполнить название и описание пулреквеста
- Нажать Create Pull Request

8. Исправления прямо на GitHub:
- Открыть файл в репозитории
- Нажать иконку редактирования
- Сделать исправления
- Создать пулреквест прямо из интерфейса GitHub

Все эти шаги были выполнены для внесения изменений в проект hexlet-components/ru-my-first-pullrequest и последующей интеграции через Pull Request.
