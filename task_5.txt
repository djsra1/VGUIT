# Основные команды Git для анализа истории изменений

## Команда git log
git log

## Просмотр изменений в коммитах
git log -p

## Команда git show
git show <commit_hash>
git show 5120bea

## Команда git blame
git blame INFO.md

## Команда git grep
git grep line
git grep -i hexlet
git grep Hexlet 5120bea
